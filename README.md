# LearnForge LMS 🎓

A complete Learning Management System with email notifications, authentication, and full-stack capabilities.

## Features

- ✅ User Authentication (JWT)
- ✅ Course Management
- ✅ Quiz System
- ✅ Discussion Forum
- ✅ Email Notifications
- ✅ Progress Tracking
- ✅ Certificates
- ✅ Payment Processing
- ✅ Admin Dashboard
- ✅ Responsive Design

## Tech Stack

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Authentication**: JWT
- **Email**: Nodemailer

## Quick Start

### Prerequisites

- Node.js v18+
- MongoDB
- Gmail account for email (or any SMTP)
