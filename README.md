## 🚀 Overview

LearnForge LMS is a comprehensive Learning Management System that enables educational institutions, instructors, and students to efficiently manage online learning activities. The platform provides a user-friendly interface for course creation, enrollment, content delivery, assessments, and progress monitoring.

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/learnforge-lms.git
```

### 2. Navigate to the Project

```bash
cd learnforge-lms
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Configure Environment Variables

Create a `.env` file in the project root.

Example:

```env
PORT=3000
MONGODB_URI=your_mongodb_connection_string
SESSION_SECRET=your_secret_key
```

### 5. Start the Application

```bash
npm start
```

or

```bash
node server/server.js
```
